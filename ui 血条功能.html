ui 血条功能

星火编辑器提供了预制 ui-分段式单位血条、预制 ui-单位血条、预制 ui-boss 血条三个显示在 ui 界面的血条功能。均需要在预制功能库中找到后应用，方可在界面编辑器中看到该血条 ui。

一、预制 ui-单位血条​

可以在预制功能的卡片面板对其进行配置

  • 主要属性是指绑定单位的生命及生命上线
  • 次要属性可以根据需要为其配置

二、预制 ui-分段式单位血条​

分段式单位血条可以配置每段血条的数值，在属性栏可进行详细配置。

三、预制 ui-boss 血条​

应用成功后可以在界面编辑器中修改血条的样式，也可以使用客户端触发语句动态的修改绑定的单位、属性等等。
